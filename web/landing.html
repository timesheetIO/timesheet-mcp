Quick Setup

Claude Code

Recommended

Run this command in your terminal:

claude mcp add timesheet -- npx -y @timesheet/mcp

Claude Desktop

Add to your claude_desktop_config.json:

"timesheet": { "command": "npx", "args": ["-y", "@timesheet/mcp"], "env": { "TIMESHEET_API_TOKEN": "your_api_token" } }
